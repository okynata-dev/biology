What this is.

Everything comes from one number — the seed. It feeds eleven morphologies, twelve stains, ten organelle slots, plus reserves, lifecycle states, and three rare anomalies. The whole thing renders in your browser. No image files. No server.

The look comes from microscope slides — bacterial staining under a light microscope. The animation is invented. Real bacteria don't actually breathe. In this archive they do.

By Timsouw. The code that renders each specimen is the same code you're looking at right now — open the page source.
